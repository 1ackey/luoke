需求文档：洛克王国“魔法学院快报”交互式数字报纸
1. 项目愿景 (Vision)
将静态的《魔法学院快报》 转化为一个充满魔法感、高交互性的单页展示站。设计风格遵循 “外简内繁”：页面排版保持报纸的严谨与简约，但每一个像素都蕴含动态反馈。

2. 视觉语言规范 (Design Language)
2.1 核心排版与颜色
基础背景：复刻 material文件夹中newspaper.jpg 的米黄羊皮纸质感（#F5F2E1），边缘带有轻微的做旧阴影。

布局结构：严格采用图片中的三栏式网格布局，使用 animal-island-ui 的圆润卡片作为模块边界，但保持极简的间距（Gap）。

字体应用：全局强制使用 “上首墩墩体”。标题采用粗体高对比色（深蓝 #2D4B73），正文采用中等字重。

边框艺术：模块使用 3px 深色实线边框，并带有 20px-30px 的不规则圆角（Animal Crossing 风格）。

2.2 资源引用
主参考图：material文件夹中newspaper.jpg

UI 库参考：animal-island-ui。

1. 极致动画方案 (The "Flashy" Animation)
3.1 预加载与开场 (The Magic Opening)
Pre-loader：屏幕中心一个 CSS 绘制的咕噜球，模拟“动森”式的落地回弹（Squash and Stretch）动画。

页面入场：报纸不是整体出现，而是像“魔法卷轴”一样从中心向两侧横向展开，或者像碎片拼接一样由中心汇聚。

3.2 交互反馈 (Micro-Interactions)
3D 摇摆卡片 (Tilt)：当鼠标移入“精灵伙伴”或“战斗帮手”版块时，整个模块随鼠标位置产生 3D 倾斜反馈。

魔法溢出效果 (Particle System)：

点击：产生向外扩散的星星粒子或“可可果”图标。

悬停：模块底部缓慢浮现一个旋转的魔法阵阴影。

流动边框：选中的版块边框会出现流动的光效，模拟魔力注入。

3.3 深度交互 (Deep Dives)
咕噜球流程图：图片中间的逻辑图需实现动态连线。能量在图标间循环流动，鼠标指到哪个环节，哪个环节就产生 Q 弹的震动。

视觉放大镜：悬停在密集文字区时，自动激活一个带圆角边框的放大镜，将内容略微放大并提升对比度。

4. 技术任务清单 (Implementation Tasks)
全局配置：

设置 animal-island-ui 的颜色变量，注入《洛克王国》主色调。

配置 @font-face 加载“上首墩墩体”。

核心组件开发：

NewspaperHeader：复刻报纸头部的魔法纹章与标题排版。

MagicCard：基于 Framer Motion 封装带有 3D Tilt 和果冻回弹效果的容器。

FloatingNotification：将右下角的“皮卡月刊”转化为悬浮的、带呼吸效果的动态组件。

自定义鼠标系统：

隐藏原生鼠标，使用animal-island-ui内涵的 Cursor样式。
